# FIELD — Landing Portfolio

Ten full brand websites for small businesses and local brands — cinematic, multi-section, motion-rich.

## Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS 4
- GSAP + ScrollTrigger + `@gsap/react`
- Premium Google fonts (Syne, Fraunces, Newsreader, Outfit, Bebas Neue, DM Sans…)

## Run

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
